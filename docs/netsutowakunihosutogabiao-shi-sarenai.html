ネットワークにホストが表示されない

2018年6月27日
By Hangil Chang

Windows 10 で、ネットワークにホストが表示されない場合の対処法。

まず、WORKGROUP が正しく設定されているか確認。

「コントロールパネル」 -> 「管理ツール」 -> 「サービス」で、

  • Function Discovery Provider Host
  • Function Discovery Resource Publication

を起動させる。
